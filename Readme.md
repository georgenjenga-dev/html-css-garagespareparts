# GARAGE SPARE PARTS 
A simple and responsive web application for browsing and purchasing automobile spare parts.
The platform allows users to explore products, contact the business, and make payments using mobile money options like M-Pesa and Airtel Money.

## Features
Home page with company overview:
 Product listing page
 Contact form with API integration
Checkout page with:
                   1.M-Pesa payment option
                   2.Airtel Money payment option
 Responsive and user-friendly UI
 JavaScript-powered interactivity
